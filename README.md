Wonder Lab is a website for a fictional community science museum aimed at children, families and school visitors.

## Description

Wonder Lab was created as my Semester Project 1. The goal was to design and build a responsive website for a science museum with a clear target audience and useful content.

The site includes information about the museum, exhibitions, events, support opportunities, and visitor-focused pages. The design is meant to feel playful and accessible while still being structured and easy to navigate.

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone https://github.com/SondreKuven/html-css-sondrekuven
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
